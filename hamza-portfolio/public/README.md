# Public Assets

Place your static assets here:

## Profile Image
- Add your profile image as `profile_img.jpeg` in this folder
- The image will be automatically picked up by the About section
- Recommended dimensions: 800x1000px (4:5 aspect ratio)
- Supported formats: JPEG, PNG, WebP

## Other Assets
- Favicon files
- Resume PDF (reference in Hero component)
- Any other static files your portfolio needs

The profile image provided in the context should be placed here before running the development server.
